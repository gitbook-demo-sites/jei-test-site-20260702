# Início Rápido

Vá do zero ao seu primeiro projeto rastreado em cerca de 5 minutos.

{% stepper %}
{% step %}
### Crie uma conta

Cadastre-se em `app.orbitly.example.com` com seu e-mail de trabalho. Toda nova conta começa no plano gratuito **Explorer** (até 3 projetos, 5 colegas de equipe).
{% endstep %}

{% step %}
### Crie seu primeiro projeto

1. Clique em **Novo Projeto** na barra lateral.
2. Dê um nome, como `Redesign do Site`.
3. Escolha um modelo. **Kanban** é um bom padrão.
{% endstep %}

{% step %}
### Adicione missões

Missões são a unidade de trabalho do Orbitly. Adicione algumas:

```
- Design do novo destaque da homepage
- Migrar blog para novo CMS
- Configurar eventos de analytics
```

Arraste missões entre colunas para atualizar seu status.
{% endstep %}

{% step %}
### Convide sua equipe

Vá em **Configurações → Membros** e convide colegas por e-mail. Funções:

| Função | Pode fazer |
| ------ | ---------- |
| Admin | Tudo, incluindo faturamento |
| Membro | Criar e editar missões |
| Visualizador | Acesso somente leitura |
{% endstep %}

{% step %}
### Defina uma janela de lançamento

Uma janela de lançamento é um sprint com tempo limitado. Clique em **Nova Janela de Lançamento**, defina um período de 2 semanas e arraste missões para dentro dela. Missões não concluídas são automaticamente transferidas quando a janela fecha.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Você está pronto quando o projeto tiver pelo menos uma missão, um colega de equipe e uma janela de lançamento.
{% endhint %}
