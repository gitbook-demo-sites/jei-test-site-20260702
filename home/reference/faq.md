# FAQ

## Who owns this site?

The test site owner is <code class="expression">space.vars.owner</code>.

## Should this site be public?

Not while it is a sandbox. Use share-link access until the content is intended for a broader audience.

## Should we keep the Docs space too?

Keep it only if it has a clear purpose. If all testing moves into the Home space, the extra Docs section can be archived or removed later.

## What should be tested first?

Start with editing, comments, change requests, preview, and publishing. Those workflows prove the core collaboration loop.

## What content should be added next?

Add a real sample project, product, or API. Real content is better for testing search, navigation, and AI answers than generic placeholder copy.
