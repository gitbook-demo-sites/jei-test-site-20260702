# Quality checklist

Use this checklist before considering a page ready.

| Category | Check |
| --- | --- |
| Purpose | The first paragraph says what the page helps with. |
| Structure | Headings follow the reader's workflow. |
| Completeness | The reader can finish the task without guessing. |
| Formatting | Steps, hints, tables, or cards are used where they improve scanning. |
| Links | Related pages are linked where the reader needs them. |
| Access | No private or sensitive information is exposed. |
| Search | The page uses natural terms and avoids internal-only shorthand. |

{% hint style="info" %}
For a test site, prioritize clarity over completeness. A small, clean site is better than a large set of placeholder pages.
{% endhint %}
