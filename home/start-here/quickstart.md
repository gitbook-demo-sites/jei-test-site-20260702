# Quickstart

Follow this flow to test the basics of the site.

{% stepper %}
{% step %}
### Edit a page

Open any page and make a small content change. Prefer a real sentence over placeholder text so search and AI answers can be tested meaningfully.
{% endstep %}

{% step %}
### Add a comment

Highlight a sentence and leave a comment that asks for a concrete improvement. This validates review collaboration.
{% endstep %}

{% step %}
### Create a change request

Group related edits into a change request so reviewers can approve the work as one unit.
{% endstep %}

{% step %}
### Preview the result

Use the site preview before publishing. Check navigation, links, and search behavior.
{% endstep %}

{% step %}
### Publish when ready

Publish only after the checklist passes. Keep share-link access enabled while this remains a sandbox.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
A good first test is to update the owner name or add one FAQ entry, then preview the result.
{% endhint %}
