# Publish checklist

Run these checks before publishing a new version of the site.

{% stepper %}
{% step %}
### Check the table of contents

Make sure every visible page belongs in the current navigation.
{% endstep %}

{% step %}
### Open key pages in preview

Test the homepage, one workflow page, one reference page, and any page with a rich block.
{% endstep %}

{% step %}
### Test links

Click all card links on the homepage and any links used in the publishing flow.
{% endstep %}

{% step %}
### Review visibility

Confirm the site is using the intended access model. For this sandbox, prefer share-link access.
{% endstep %}

{% step %}
### Publish and verify

Publish, open the shared URL in a clean browser session, and confirm the expected pages are visible.
{% endstep %}
{% endstepper %}
