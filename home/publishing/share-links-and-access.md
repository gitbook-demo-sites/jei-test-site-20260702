# Share links and access

This test site should stay easy to share without becoming fully public.

## Recommended setup

| Setting | Recommendation |
| --- | --- |
| Site visibility | Share-link while testing. |
| Search indexing | Keep public indexing off until the content is production-ready. |
| Member access | Admins for setup, editors for content contributors, commenters for reviewers. |
| External review | Share a specific link and ask for feedback on a focused page. |

{% hint style="info" %}
Share-link access is useful for demos and internal testing because it gives reviewers a stable URL without making the site broadly discoverable.
{% endhint %}
