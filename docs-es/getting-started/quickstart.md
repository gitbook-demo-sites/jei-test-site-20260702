# Inicio rápido

Pasa de cero a tu primer proyecto rastreado en aproximadamente 5 minutos.

{% stepper %}
{% step %}
### Crea una cuenta

Regístrate en `app.orbitly.example.com` con tu correo laboral. Cada cuenta nueva comienza con el plan gratuito **Explorer** (hasta 3 proyectos, 5 compañeros).
{% endstep %}

{% step %}
### Crea tu primer proyecto

1. Haz clic en **Nuevo Proyecto** en la barra lateral.
2. Asígnale un nombre, como `Rediseño del sitio web`.
3. Elige una plantilla. **Kanban** es una buena opción por defecto.
{% endstep %}

{% step %}
### Añade misiones

Las misiones son la unidad de trabajo en Orbitly. Añade algunas:

```
- Diseñar el nuevo héroe de la página principal
- Migrar el blog al nuevo CMS
- Configurar eventos de analítica
```

Arrastra las misiones entre columnas para actualizar su estado.
{% endstep %}

{% step %}
### Invita a tu equipo

Ve a **Configuración → Miembros** e invita a compañeros por correo electrónico. Roles:

| Rol | Puede hacer |
| ---- | ------ |
| Admin | Todo, incluyendo facturación |
| Miembro | Crear y editar misiones |
| Espectador | Acceso solo lectura |
{% endstep %}

{% step %}
### Establece una ventana de lanzamiento

Una ventana de lanzamiento es un sprint con tiempo limitado. Haz clic en **Nueva ventana de lanzamiento**, establece un rango de 2 semanas y arrastra misiones dentro. Las misiones no terminadas se trasladan automáticamente cuando la ventana cierra.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Estarás listo cuando el proyecto tenga al menos una misión, un compañero y una ventana de lanzamiento.
{% endhint %}
